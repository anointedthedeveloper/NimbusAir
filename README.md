# NimbusAir

NimbusAir is a flight booking platform concept built with Next.js and styled as a premium airline product demo.

## What is included

- A polished home page with search, highlights, and stack overview
- Search, booking, dashboard, admin, and flight status pages
- SEO metadata, sitemap, robots file, and Open Graph image
- Flight-themed UI with icons and a dark premium visual system
- Creator and sponsorship credit for anointedthedeveloper and anobyte

## Brand credits

## Run locally

```bash
npm install
npm run dev
```

## Notes

- The backend is presented as a product-ready concept layer.
- The frontend is built to make future FastAPI, PostgreSQL, JWT, and payment integrations easy to add.
